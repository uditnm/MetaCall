1. Go to Install and run "MetaVcam.bat" as administrator to install a virtual
   camera named "MetaCall".

2. Once the installation is complete please restart the webrtc app or entire 
   computer in order for the changes to take place.

3. This app requires python. Install python-3.10.10.exe if you have 32 bit system or python-3.10.10-amd64.exe
   if youn have a 64 bit system with pip option selected.

4. After that run Meta-Ava.exe.

4. Then open the webrtc based site or app and goto video source in settings.

5. Select MetaCall from the listed video sorces. 

Note:- Please install python from the Install folder only as this project is compatible with python version-3.10 and older.
